GNS3
====

Network Simulator. 

Install:
-------

    # Ubuntu install:
    sudo add-apt-repository ppa:gns3/ppa
    sudo apt update                                
    sudo apt install gns3-gui gns3-server
    # “sudo usermod -aG group user_name” 
    ubridge libvirt kvm wireshark docker
