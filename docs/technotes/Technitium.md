Technitium
=====

Technitium is a secure by defualt DNS forwarder. Blocks malicous domains , ads, and encrypts DNS traffic. 

Install
-------

     curl -sSL https://download.technitium.com/dns/install.sh | sudo bash
     # http://localhost:5380/
     User: admin
     Pass: admin
