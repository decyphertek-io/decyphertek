# DecypherTek AI

Linux based decyphertek.ai CLI chat. Modular stores: mcp ; agent ; app , to customize AI . 

## Features

- **AI Chat** - Generalized / Specialized AI chat.
- **Rag Chat** - Upload docs and inquire about them. 
- **App Store** - Python apps.
 **Agent Store** - Manage the AI personality.
- **MCP Servers** - Modular AI Skills.

## Quick Start
```
# Install 
curl -fsSL https://raw.githubusercontent.com/decyphertek-io/decyphertek-ai/main/install.sh | bash
# Run
decyphertek.ai
# help
/help
# Uninstall
curl -fsSL https://raw.githubusercontent.com/decyphertek-io/decyphertek-ai/main/uninstall.sh | bash
```
