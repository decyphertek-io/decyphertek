Balena Etcher
=====

An easy way to create live USB.  

Install
-------

     $ curl -1sLf 'https://dl.cloudsmith.io/public/balena/etcher/setup.deb.sh' | sudo -E bash
     $ sudo apt-get update 
     $ sudo apt-get install balena-etcher-electron
